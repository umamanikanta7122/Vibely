# Vibely

Vibely is a social media web application inspired by Instagram, built using Django.

Features:
- User Authentication
- Posts (Image/Video)
- Stories with auto-switch
- Reels Section
- Likes, Comments, Views
- Search Users
- Follow / Unfollow
- Profile Management
- Multi-story support
- Responsive UI

Tech Stack:
- Django
- HTML
- CSS
- JavaScript
- SQLite
